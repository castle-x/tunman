# Security Policy

## Supported Versions

This project is currently in active development. Security fixes are provided for the latest code on the `main` branch.

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue.

Please report it privately to the maintainers through GitHub Security Advisories:

- Go to the repository's **Security** tab
- Click **Report a vulnerability**
- Provide reproduction steps and impact details

We will acknowledge reports as soon as possible and coordinate a fix and disclosure timeline.
